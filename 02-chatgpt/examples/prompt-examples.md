# ChatGPT Prompt Examples

A collection of ready-to-use prompts demonstrating effective use of ChatGPT.

---

## 1. Writing Prompts

### Professional Email
```
Write a professional email to my team announcing a 2-week project deadline extension.
The reason is unexpected technical issues with the database migration.
Keep the tone positive and end with next steps.
```

### Blog Post
```
Write a 500-word introductory blog post about the benefits of learning to use AI tools
in everyday work. Target audience: non-technical professionals. Use a friendly,
encouraging tone. Include a short introduction, 3 main points, and a conclusion.
```

### Resume Bullet Point
```
Rewrite this resume bullet point to be more impactful and results-focused:
"Helped customers with their questions and solved problems."
```

---

## 2. Learning Prompts

### Explain Like I'm a Beginner
```
Explain how neural networks work. Assume I have no technical background.
Use a simple analogy to make it easy to understand.
```

### Step-by-Step Guide
```
Create a step-by-step beginner's guide for using Excel pivot tables.
Include what they are, why they're useful, and how to create one from scratch.
```

### Quiz Me
```
I'm studying [topic]. Create 5 multiple-choice questions to test my knowledge.
After I answer, tell me which answers are correct and explain why.
```

## 3. Coding Prompts

### Write a Function
```
Write a Python function called `calculate_bmi` that takes weight (kg) and height (m)
as parameters and returns the BMI value and category (Underweight, Normal, Overweight, Obese).
Include docstrings and example usage.
```

### Debug Code
```
This Python code is supposed to sort a list of dictionaries by the "age" key,
but it throws an error. Find and fix the bug:

data = [{"name": "Alice", "age": 30}, {"name": "Bob"}, {"name": "Carol", "age": 25}]
sorted_data = sorted(data, key=lambda x: x["age"])
print(sorted_data)
```

### Code Review
```
Review this JavaScript function for readability, performance, and potential bugs.
Suggest improvements with explanations:

[paste your code here]
```

## 4. Analysis Prompts

### Pros and Cons
```
Give me a balanced analysis of the pros and cons of working remotely full-time.
Format as two lists: Pros and Cons, with 5 items each.
```

### Compare Options
```
Compare these three cloud providers for a startup: AWS, Google Cloud, and Azure.
Focus on pricing, ease of use, and startup programs. Format as a comparison table.
```

### Summarize
```
Summarize the following article in 5 bullet points. Focus on the key facts and
any actionable takeaways:

[paste article text here]
```
