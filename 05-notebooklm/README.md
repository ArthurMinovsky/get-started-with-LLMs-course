# NotebookLM — เริ่มต้นใช้งาน

## NotebookLM คืออะไร?

**NotebookLM** คือเครื่องมือ AI สำหรับงานวิจัยและจดโน้ตจาก **Google** ออกแบบมาเพื่อช่วยทำความเข้าใจเอกสารของคุณเองโดยเฉพาะ

จุดเด่นคือ **ตอบจากแหล่งข้อมูลที่คุณอัปโหลดเป็นหลัก** พร้อมการอ้างอิงกลับไปยังต้นทาง ทำให้เชื่อถือได้มากขึ้นสำหรับงานวิจัย

## สิ่งที่ทำให้ NotebookLM ต่างจาก LLM ทั่วไป

| คุณสมบัติ | LLM ทั่วไป | NotebookLM |
|---------|------------------------------|------------|
| แหล่งความรู้ | ข้อมูลฝึกทั่วไป | แหล่งข้อมูลที่คุณอัปโหลด |
| ความเสี่ยง Hallucination | สูงกว่า | ต่ำกว่า (ยึดตามแหล่งข้อมูล) |
| การอ้างอิง | ไม่สม่ำเสมอ | มีการอ้างอิงชัดเจน |
| เหมาะกับงาน | ถามตอบทั่วไป/เขียน | วิจัย/วิเคราะห์เอกสาร |

## ประเภทแหล่งข้อมูลที่รองรับ

- Google Docs / Slides
- PDF
- URL เว็บไซต์
- ไฟล์ข้อความ/Markdown
- YouTube URL
- ไฟล์เสียง

## เริ่มต้นใช้งาน

1. เข้า [https://notebooklm.google.com](https://notebooklm.google.com)
2. สร้าง **New Notebook**
3. กด **Add sources** เพื่อเพิ่มไฟล์หรือ URL
4. เริ่มถามคำถามจากเอกสารที่อัปโหลด

## ฟีเจอร์เด่น

### Chat กับเอกสารของคุณ
- ถามเชิงลึกจากเนื้อหาในเอกสาร
- คำตอบมี citation ให้ตรวจสอบได้

### Notebook Guide
- สารบัญ, สรุปแบบเรียน, FAQ, ไทม์ไลน์, executive summary

### Audio Overview
- สร้างสรุปแบบพอดแคสต์ด้วยเสียง AI

### Source Annotations
- กด citation เพื่อไปยังข้อความต้นทางที่ถูกอ้างอิง

## กรณีใช้งาน

- งานวิจัยเชิงวิชาการ
- เตรียมสอบและทำสรุป
- วิเคราะห์รายงานธุรกิจ
- ทบทวนเอกสารกฎหมาย
- เตรียมข้อมูลก่อนเขียนบทความ

## เคล็ดลับ

1. เริ่มจาก Notebook Guide ก่อน
2. ตรวจ citations ทุกครั้ง
3. ใช้คำถามเฉพาะเจาะจง
4. ผสมหลายแหล่งข้อมูลเพื่อมุมมองที่ครบขึ้น
5. ใช้ Audio Overview สำหรับทบทวนเร็ว

## ข้อจำกัด

- ตอบได้ดีเมื่อมีแหล่งข้อมูลที่อัปโหลด
- จำกัดจำนวนแหล่งข้อมูลต่อโน้ตบุ๊ก
- ไม่ดึงข้อมูลเว็บแบบเรียลไทม์โดยอัตโนมัติ
- ประสิทธิภาพดีที่สุดกับภาษาอังกฤษ (แต่รองรับภาษาอื่น)
