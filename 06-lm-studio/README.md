# LM Studio — เริ่มต้นใช้งาน

## LM Studio คืออะไร?

**LM Studio** คือแอปเดสก์ท็อปที่ช่วยให้คุณดาวน์โหลดและรันโมเดล LLM แบบโอเพนซอร์สบนเครื่องของตัวเองได้แบบออฟไลน์ โดยไม่ต้องส่งข้อมูลไปยังเซิร์ฟเวอร์ภายนอก

เหมาะกับงานที่ต้องการความเป็นส่วนตัวหรือไม่ต้องการพึ่งพา API ภายนอก

## ข้อดีของการรัน LLM แบบโลคัล

| ข้อดี | คำอธิบาย |
|-----------|-------------|
| **Privacy** | ข้อมูลอยู่ในเครื่องคุณ |
| **Cost** | ไม่มีค่า API รายครั้ง |
| **Offline** | ใช้งานได้แม้ไม่มีอินเทอร์เน็ต |
| **Customization** | เลือกโมเดลและตั้งค่าได้อิสระ |
| **Unlimited Usage** | ไม่ติดโควตาเรียกใช้งาน |

## ความต้องการของระบบ (ย่อ)

- OS: Windows/macOS/Linux
- RAM: อย่างน้อย 8GB (แนะนำ 16GB+)
- พื้นที่เก็บข้อมูล: อย่างน้อย 10–50GB
- GPU ช่วยให้เร็วขึ้นมาก (ถ้ามี)

## วิธีติดตั้งและใช้งาน

### ขั้นที่ 1 — ดาวน์โหลด
1. เข้า [https://lmstudio.ai](https://lmstudio.ai)
2. ดาวน์โหลดตามระบบปฏิบัติการ
3. ติดตั้งตามขั้นตอน

### ขั้นที่ 2 — เลือกและดาวน์โหลดโมเดล
1. เปิดแท็บ **Search**
2. ค้นหาโมเดล เช่น `llama`, `mistral`, `gemma`
3. เลือกขนาดโมเดลให้เหมาะกับ RAM/GPU
4. ดาวน์โหลด (ไฟล์ส่วนใหญ่เป็น GGUF)

### ขั้นที่ 3 — เริ่มแชต
1. ไปที่แท็บ **Chat**
2. เลือกโมเดลที่ดาวน์โหลดแล้ว
3. รอโหลดโมเดลและเริ่มพิมพ์ข้อความ

## แนวคิดเรื่อง Quantization

Quantization ช่วยลดขนาดโมเดลและการใช้หน่วยความจำ แลกกับคุณภาพบางส่วน:

| Quantization | ขนาดไฟล์ | คุณภาพ | เหมาะเมื่อ |
|-------------|-----------|---------|----------|
| **Q2_K** | เล็กสุด | ต่ำกว่า | RAM จำกัดมาก |
| **Q4_K_M** | กลาง | ดี | สมดุลดีที่สุด (แนะนำ) |
| **Q5_K_M** | ใหญ่ขึ้น | ดีกว่า | มี RAM มากขึ้น |
| **Q8_0 / F16** | ใหญ่มาก | สูง | เน้นคุณภาพสูงสุด |

## ฟีเจอร์สำคัญ

- แชตหลายรอบพร้อม context
- ตั้ง system prompt และพารามิเตอร์ได้
- เปิด Local Server แบบ OpenAI-compatible ได้ (`http://localhost:1234/v1`)
- จัดการโมเดลที่ดาวน์โหลดได้สะดวก

## ข้อจำกัด

- คุณภาพ/ความเร็วขึ้นกับฮาร์ดแวร์
- มักช้ากว่า cloud API
- ไม่มีการท่องเว็บโดยตรง
- ต้องดูแลอัปเดตโมเดลเอง
