# Gemini — เริ่มต้นใช้งาน

## Gemini คืออะไร?

**Gemini** คือชุดโมเดล AI แบบหลายสื่อ (multimodal) จาก **Google DeepMind** รองรับข้อความ โค้ด รูปภาพ เสียง และวิดีโอ

Gemini ถูกผสานเข้ากับระบบของ Google อย่างแนบแน่น เช่น Search, Workspace และ Android

## โมเดล Gemini

| โมเดล | คำอธิบาย | เหมาะกับงาน |
|-------|-------------|----------|
| **Gemini 1.5 Flash** | เร็วและประหยัด | งานทั่วไป/ตอบไว |
| **Gemini 1.5 Pro** | ความสามารถสูง, context ยาว | งานซับซ้อน/เอกสารยาว |
| **Gemini Ultra** | ทรงพลังที่สุด | วิจัยและวิเคราะห์ระดับสูง |
| **Gemini Nano** | ทำงานบนอุปกรณ์ | งานมือถือ |

## แพ็กเกจการใช้งาน

| แพ็กเกจ | การเข้าถึง | ฟีเจอร์ |
|------|--------|----------|
| **Free** | gemini.google.com | ฟีเจอร์มาตรฐาน |
| **Gemini Advanced** | gemini.google.com | เข้าถึงรุ่นความสามารถสูง |
| **Google Workspace** | workspace.google.com | ใช้ใน Docs/Sheets/Gmail |
| **API (AI Studio)** | aistudio.google.com | ทดลอง API สำหรับนักพัฒนา |
| **Vertex AI** | cloud.google.com | ใช้ระดับองค์กร |

## เริ่มต้นใช้งาน

1. เข้า [https://gemini.google.com](https://gemini.google.com) แล้วล็อกอิน
2. นักพัฒนาลอง [https://aistudio.google.com](https://aistudio.google.com) เพื่อทดลอง API
3. หากใช้ Workspace ให้มองหาปุ่ม Gemini ใน Gmail/Docs/Sheets/Slides

## ฟีเจอร์เด่น

### ความเข้าใจหลายสื่อ
Gemini วิเคราะห์ได้ทั้งข้อความ รูป เสียง วิดีโอ และโค้ดในพรอมป์เดียว

### การเชื่อมต่อระบบ Google
- Gmail: ร่าง/สรุปอีเมล
- Docs: เขียน/ปรับปรุงเนื้อหา
- Sheets: วิเคราะห์ข้อมูลและช่วยเขียนสูตร
- Slides: ช่วยร่างงานนำเสนอ

### Extensions
เชื่อมกับบริการต่าง ๆ เช่น Google Search, YouTube, Maps และบริการเดินทาง

### Gemini API & AI Studio
- ได้ API key สำหรับทดลอง
- ทดสอบ system instruction
- ทดสอบอินพุตแบบหลายสื่อ

## กรณีใช้งานทั่วไป

| งาน | ตัวอย่างพรอมป์ |
|----------|---------------|
| วิเคราะห์รูปภาพ | “ในภาพนี้มีอะไรบ้าง” |
| สรุปวิดีโอ | “สรุปประเด็นสำคัญจากวิดีโอนี้” |
| สร้างโค้ด | “เขียนฟังก์ชัน JavaScript เรียงข้อมูลตามวันที่” |
| เขียนอีเมล | “เขียนอีเมลขอโทษอย่างมืออาชีพ” |
| วางแผนท่องเที่ยว | “ช่วยจัดทริปโตเกียว 5 วัน” |

## เคล็ดลับ

1. เปิด Extensions เมื่อต้องการข้อมูลล่าสุด
2. รวมหลายชนิดไฟล์ในพรอมป์เดียว
3. ใช้ใน Workspace เพื่อทำงานได้เร็วขึ้น
4. ทดลองโมเดลต่าง ๆ ผ่าน AI Studio
