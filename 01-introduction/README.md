# บทนำสู่ Large Language Models (LLMs)

## LLM คืออะไร?

**Large Language Model (LLM)** คือโมเดลปัญญาประดิษฐ์ที่ถูกฝึกจากข้อมูลข้อความจำนวนมหาศาล โมเดลจะเรียนรู้รูปแบบของภาษา และสามารถสร้าง สรุป แปล และให้เหตุผลกับข้อความได้อย่างมีประสิทธิภาพ

LLM คือเทคโนโลยีเบื้องหลังเครื่องมืออย่าง ChatGPT, Claude และ Gemini โดยอาศัยสถาปัตยกรรมโครงข่ายประสาทที่เรียกว่า **Transformer** จากงานวิจัยสำคัญปี 2017 *"Attention Is All You Need"*

## LLM ทำงานอย่างไร (ภาพรวม)

1. **Training**: ฝึกโมเดลด้วยเอกสารจำนวนมหาศาลจากอินเทอร์เน็ต หนังสือ และแหล่งข้อมูลอื่น
2. **Tokenization**: แบ่งข้อความเป็นหน่วยย่อยที่เรียกว่า *token*
3. **Prediction**: เรียนรู้การทำนาย token ถัดไปจากบริบทก่อนหน้า
4. **Fine-tuning & RLHF**: ปรับจูนหลังการฝึกด้วยฟีดแบ็กจากมนุษย์ เพื่อให้คำตอบมีประโยชน์และปลอดภัยมากขึ้น

## แนวคิดสำคัญ

| คำศัพท์ | ความหมาย |
|------|------------|
| **Prompt** | ข้อความอินพุตที่คุณป้อนให้ LLM |
| **Completion / Response** | ข้อความผลลัพธ์ที่โมเดลสร้าง |
| **Token** | หน่วยข้อความย่อย (เฉลี่ย ~0.75 คำ) |
| **Context Window** | ปริมาณข้อความสูงสุดที่โมเดลประมวลผลได้ในครั้งเดียว |
| **Temperature** | ระดับความสุ่มของคำตอบ (0 = ตายตัว, 1 = สร้างสรรค์) |
| **System Prompt** | คำสั่งตั้งต้นที่ให้โมเดลก่อนเริ่มสนทนา |
| **Hallucination** | โมเดลให้ข้อมูลผิดแต่ฟังดูมั่นใจ |
| **Fine-tuning** | ฝึกโมเดลเพิ่มด้วยชุดข้อมูลเฉพาะงาน |
| **RAG** | Retrieval-Augmented Generation — ผสาน LLM กับแหล่งความรู้ภายนอก |

## ประเภทของเครื่องมือ LLM

### แบบคลาวด์ (API / เว็บ)
- รันบนเซิร์ฟเวอร์ของผู้ให้บริการ
- ตัวอย่าง: ChatGPT, Claude, Gemini
- ข้อดี: ไม่ต้องมีฮาร์ดแวร์แรง อัปเดตตลอด
- ข้อจำกัด: ต้องใช้อินเทอร์เน็ต และข้อมูลส่งไปยังผู้ให้บริการ

### แบบโลคัล / บนอุปกรณ์
- รันบนเครื่องของคุณเองทั้งหมด
- ตัวอย่าง: LM Studio (ร่วมกับโมเดลโอเพนซอร์ส)
- ข้อดี: เป็นส่วนตัว ใช้งานออฟไลน์ได้ ไม่เสียค่า API
- ข้อจำกัด: ต้องใช้ฮาร์ดแวร์ที่เหมาะสม (แนะนำ GPU)

### แบบเวิร์กโฟลว์และระบบอัตโนมัติ
- เครื่องมือที่นำ LLM ไปเชื่อมกับกระบวนการอัตโนมัติ
- ตัวอย่าง: n8n, LangChain, Flowise
- ข้อดี: ขยายระบบได้ง่าย ทำงานซ้ำได้เป็นมาตรฐาน
- ข้อจำกัด: ต้องมีการตั้งค่าทางเทคนิคพอสมควร
