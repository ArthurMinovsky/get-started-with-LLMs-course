# LLM Glossary — Quick Reference

Use this glossary as a reference while going through the course.

| Term | Definition |
|------|------------|
| **LLM** | Large Language Model — an AI trained on large text datasets |
| **Prompt** | The text input you give to an AI model |
| **Token** | A unit of text (~0.75 words); models process and price by tokens |
| **Context Window** | Maximum tokens the model can process in one session |
| **Temperature** | Controls creativity (0 = focused, 1 = creative/random) |
| **System Prompt** | Hidden instructions that define the AI's behavior |
| **Hallucination** | AI generating confident but incorrect information |
| **Fine-tuning** | Additional training of a model on specific data |
| **RAG** | Retrieval-Augmented Generation — AI + external knowledge base |
| **Embedding** | Numerical representation of text for similarity search |
| **Vector DB** | Database storing embeddings for semantic search |
| **RLHF** | Reinforcement Learning from Human Feedback — training technique |
| **Zero-shot** | Asking the AI to do a task without examples |
| **Few-shot** | Providing a few examples before the actual task |
| **Chain-of-Thought** | Asking AI to reason step-by-step before answering |
| **Inference** | Running a trained model to generate responses |
| **Quantization** | Reducing model precision to decrease size and memory use |
| **GGUF** | File format for quantized LLMs (used by LM Studio) |
| **API** | Application Programming Interface — programmatic access to a service |
| **Multimodal** | AI that handles multiple input types (text, image, audio, video) |
